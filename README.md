# 🍵 校园咖啡馆数智化敏捷架构沙盘系统

> HRM课堂沙盘实战 | 分布式决策 + 集中式实时推演

## 快速启动

```bash
# 安装依赖
npm install

# 启动服务器（默认端口3000）
npm start

# Mock演示模式（大屏自动滚动数据，无需学生端）
npm start -- --mock
```

启动后访问：
- 🎓 **学生H5端**: http://localhost:3000/student
- 🖥️ **教师大屏**: http://localhost:3000/dashboard

## 系统架构

```
手机H5端（学生）           后端服务器              Web大屏（教师）
    │                        │                      │
    ├─ POST /api/team/init  ─┤                      │
    ├─ POST /api/team/submit─┤── 判定引擎 ──────────┤
    │                        │   ├ 管理幅度击穿检测   │
    │                        │   ├ 矩阵重组评估       │
    │                        │   └ 危机压力测试       │
    │                        │                      │
    └─ WebSocket ◀──────────┴── 实时广播 ──────────┤
                                                 ├ Echarts雷达图
                                                 ├ 架构散点图
                                                 └ 滚动战报
```

## 判定引擎规则

| 场景 | 条件 | 生存率 |
|------|------|--------|
| 管理幅度击穿 | 任意节点 span > 8 | ≤40% |
| 矩阵重组+通过危机 | ≥3组 + 私域组≥2人 | 95% |
| 矩阵重组+未通过危机 | ≥3组 + 无/弱私域组 | 35% |
| 未重组+通过危机 | <3组 + 私域组≥2人 | 70% |
| 未重组+未通过危机 | <3组 + 无/弱私域组 | 10% |

## 技术栈

- **后端**: Node.js + Express + Socket.io + SQLite
- **前端**: 原生HTML5/CSS3/JS + Canvas拓扑编辑器 + Echarts可视化
- **核心特性**:
  - 移动端触摸拖拽拓扑编辑
  - 60秒危机倒计时压力测试
  - Mock保底模式（断网/演示一键切换）
  - 3秒轮询降级方案（WebSocket失败时）

## 教师操作

| 按钮 | 功能 |
|------|------|
| 🎭 Mock演示 | 开启/关闭自动数据演示模式 |
| ⚡ 触发危机 | 向全班学生端广播危机事件 |
| ↻ 重置 | 清空本轮所有数据 |
